# Getting Started with CloudToLocalLLM

Welcome to CloudToLocalLLM! This guide will help you get up and running quickly.

## Prerequisites

Before you begin, ensure you have:

- Flutter SDK 3.8+
- Node.js 16/18+
- Ollama (optional, for local models)

For detailed prerequisites, see [Prerequisites](PREREQUISITES.md).

## Quick Installation

1. Install Dependencies:
```powershell
flutter pub get  # Flutter dependencies
npm install     # Node.js dependencies
```

2. Start Development Environment:
```powershell
# Terminal 1: Flutter UI
flutter run -d windows  # or -d chrome for web

# Terminal 2: Backend
npm run dev
```

For detailed installation instructions, see [Installation Guide](INSTALLATION.md).
